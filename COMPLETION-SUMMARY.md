## How To Use This QMS

### For Production Crew
1. **Access Work Instructions** (WI-101 through WI-108)
   - One for each of 8 production phases
   - Visual ASCII diagrams guide assembly steps
   - Tools and equipment lists provided
   - Common mistakes and prevention tips included

2. **Follow Hold Point Inspections**
   - Check FORM-I101 through FORM-I108 at each gate
   - Pass/fail decision required before moving to next phase
   - Corrective actions documented on form if needed

### For Supervisors & QA
1. **Review Relevant SOPs**
   - SOP-011 for planning and scheduling
   - SOP-013 for hold point management
   - SOP-014 for module traveler tracking
   - SOP-016 for metrics and reporting

2. **Conduct Inspections**
   - Use FORM-I101-108 at each HP
   - Document measurements and observations
   - Authorize pass/fail/conditional decisions

### For Management
1. **Reference Complete Procedures** for any manufacturing function
2. **Track Quality Metrics** using SOP-016 dashboard
3. **Manage Changes** using SOP-005 change management
4. **Review Compliance** using SOP-009 for CDOH requirements

### For Compliance/Regulatory
1. **Access QMS Structure** via QMS-Index.md
2. **Review Regulatory Mapping** in SOP-009
3. **Audit Using** SOP-003 procedures
4. **Verify Traceability** via module travelers and data plates
