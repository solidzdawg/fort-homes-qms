# PHASE 05: Insulation

## Overview
Covers installation of thermal and acoustic insulation, air sealing, and vapor barriers. Includes QC Hold #5 and TPIA Hold #4.

## Key Steps
- Install insulation per plans and code
- Air sealing and vapor retarder installation
- Inspection for gaps, compression, and coverage
- Internal QC Hold #5
- TPIA Hold #4
