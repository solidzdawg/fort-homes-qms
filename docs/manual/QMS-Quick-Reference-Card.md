## SOP Section Order (Required)

```
1. Title & Document Control Header
2. Purpose (1-2 paragraphs)
3. Scope (bullet list)
4. References & Standards
5. Roles & Responsibilities (table)
6. Materials & Equipment
7. Pre-Production Checklist
8. Detailed Procedure (numbered steps)
9. Quality Acceptance Criteria
10. Nonconforming Items
11. Work Instructions (cross-references)
12. Hold Point Inspection
13. Records & Documentation
14. Training & Competency
15. Safety & Environment
16. Appendices
17. Approval Signatures
18. Revision History
```

## Quality Standards

| Element | Standard |
| :--- | :--- |
| **Body Font Size** | 11pt (Calibri or Arial) |
| **Line Spacing** | 1.15 |
| **Margins** | 0.75" all sides |
| **Headings** | Bold, Color-coded (Navy/Teal) |
| **White Space** | Minimum 15% margin/padding |
| **Tables** | Black borders, header row bold |
| **Images** | 150 DPI minimum resolution |
| **Hyperlinks** | Blue, underlined, tested |

## Before Publishing Any Document

- [ ] Unique document ID (no duplicates)
- [ ] Current revision number (1.0 for new)
- [ ] Effective date filled in
- [ ] Process owner identified
- [ ] All links tested and working
- [ ] Signatures/approvals obtained
- [ ] Scanned and filed in system
- [ ] Team notified of change
- [ ] Old versions archived

## Key QMS Principles

1. **Clear Hierarchy:** Main document â†’ Details â†’ Records
2. **Visual First:** 60% diagrams/photos, 40% text
3. **Actionable Steps:** Numbered sequences, not paragraphs
4. **Measurable Criteria:** Tolerances, pass/fail ranges specified
5. **Accountability:** Roles, signatures, approval trails
6. **Traceability:** Every step documented and traceable
