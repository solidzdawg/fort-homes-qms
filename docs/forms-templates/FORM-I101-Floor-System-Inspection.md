## Quality Acceptance Criteria

**PASS Criteria (all must be met):**
- [ ] All visual inspection items = PASS
- [ ] All measurements within tolerance
- [ ] No structural defects
- [ ] Proper fastening pattern (100% per SOP-101)
- [ ] Clean, debris-free surface
- [ ] Floor moisture ≤13%

**FAIL Criteria (any one causes rejection):**
- [ ] Cracks or structural damage
- [ ] Out-of-spec dimensions
- [ ] Missing or improper fasteners
- [ ] Moisture >13%
- [ ] Incomplete sealing
- [ ] Surface contamination
