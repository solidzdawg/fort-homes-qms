## Quality Acceptance Criteria

**PASS Criteria (all must be met):**
- [ ] All mechanical items properly installed and sealed

- [ ] Plumbing lines properly sized, pitched, and supported
- [ ] All tests passed
- [ ] No Code violations
- [ ] Proper clearances maintained

**FAIL Criteria (any one causes rejection):**
- [ ] Code violations present
- [ ] Improper sizing or spacing
- [ ] Test failures
- [ ] Inadequate support or damage
- [ ] Improper pitch or routing
- [ ] Missing components
