## Quality Acceptance Criteria

**PASS Criteria (all must be met):**
- [ ] All visual inspection items = PASS
- [ ] All measurements within tolerance
- [ ] Walls plumb and square per specifications
- [ ] Proper spacing and fastening pattern
- [ ] All openings sized correctly
- [ ] Sheathing fully attached and fastened per SOP-102

**FAIL Criteria (any one causes rejection):**
- [ ] Structural damage or misalignment
- [ ] Out-of-spec spacing or dimensions
- [ ] Missing or improper fasteners (>2% deficiency)
- [ ] Bowing, twisting, or plumb issues >1/8" per 10'
- [ ] Incomplete sheathing or fastening
- [ ] Contamination or debris
