## Quality Acceptance Criteria

**PASS Criteria (all must be met):**
- [ ] All visual inspection items = PASS
- [ ] All measurements within tolerance
- [ ] Roof pitch correct per design
- [ ] Proper spacing and fastening per SOP-103
- [ ] All sheathing fully attached
- [ ] No structural damage or defects

**FAIL Criteria (any one causes rejection):**
- [ ] Structural damage, cracks, or separation
- [ ] Out-of-spec pitch or dimensions
- [ ] Missing or improper fasteners (>2%)
- [ ] Incomplete sheathing
- [ ] Ventilation blocked
- [ ] Safety hazards present
