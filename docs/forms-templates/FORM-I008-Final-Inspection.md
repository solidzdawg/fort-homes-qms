| B2.2 | Depressurization to 50 Pascals achieved | [ ] | Target: -50 Pa Actual: _____ Pa |
| B2.3 | CFM50 measured (airflow at 50 Pa) | [ ] | CFM50: _____ |
| B2.4 | Building volume calculated | [ ] | Volume: _____ cubic feet |
| B2.5 | ACH50 calculated (CFM50 ÷ Volume × 60) | [ ] | **ACH50: _____** |
| B2.6 | IHIP target achieved: ACH50 ≤3.0 | [ ] | **PASS / FAIL** |
| B2.7 | Test conditions documented (temp, wind, weather) | [ ] | Temp: _____ Wind: _____ |

**ACH50 Calculation Example:**
- CFM50: 150 CFM
- Volume: 12,000 cubic feet
- ACH50 = (150 ÷ 12,000) × 60 = 0.75 ACH50 ✓ PASS

**PASS CRITERIA:** ACH50 ≤ 3.0 (IHIP requirement)
**Fort Homes Target:** ACH50 ≤ 2.0 (exceeds IHIP)

**If FAIL (ACH50 > 3.0):**
- Identify leakage points using blower door running (smoke pencil, hand test)
- Document leakage locations
- Issue NCR, perform corrective air sealing
- Retest until PASS
