## Quality Acceptance Criteria

**PASS Criteria (all must be met):**
- [ ] All drywall application items = PASS
- [ ] All taping and finishing items = PASS
- [ ] All measurements within tolerance
- [ ] No damage, water stains, or defects
- [ ] All joints properly taped and finished
- [ ] Surface ready for paint/finish

**FAIL Criteria (any one causes rejection):**
- [ ] Missing or improper fasteners (>2%)
- [ ] Fastener spacing out of spec
- [ ] Cracks, gouges, or water damage
- [ ] Improper joint finishing
- [ ] Surface not ready for paint
- [ ] Fasteners not properly set
