## 1. Purpose

To establish procedures for material receiving inspection, lot identification, and traceability tracking. This SOP ensures only approved materials enter production, maintains lot segregation, and enables rapid identification in case of supplier issues or field failures.

---

## 2. Scope

This procedure applies to:
- All incoming materials and components
- Supplier receiving inspection and acceptance
- Lot numbering and identification
- Material traceability through production
- Non-conforming material disposition

**Applies to:** Receiving Department, Materials Manager, Quality Inspector, Production Supervisor

## 5. Receiving Inspection Process

### 5.1 Receipt Verification
- Verify shipment against purchase order
- Check package condition and integrity
- Confirm supplier certification documentation
- Identify defects or damage immediately

### 5.2 Lot Assignment & Labeling
- Assign unique lot number to each material batch
- Include supplier lot/batch number reference
- Record date of receipt and expiration dates
- Apply traceability labels to all containers

---

## 6. Traceability Documentation

### 6.1 Lot Tracking System
- Maintain lot registry with supplier, date, quantity received
- Link lots to production modules via traveler documentation
- Record lot consumption date and location
- Track lot status: received, in-production, consumed, archived

### 6.2 Non-Conforming Material Handling
- Quarantine non-conforming materials immediately
- Document inspection findings and disposition
- Obtain Quality Manager approval for use or return
- Investigate supplier quality issues per SOP-004

---

## 7. Lot Recall & Traceability

Upon supplier or field-quality issue, use lot traceability database to identify affected modules. Production Supervisor notifies Quality Manager. Affected modules flagged for rework or customer notification per SOP-018.
