# WI-002B: Subfloor Installation

## Step-by-Step Instructions
1. Apply construction adhesive to joist tops.
2. Lay subfloor panels perpendicular to joists, stagger joints.
3. Fasten panels per schedule (e.g., 6" edge, 12" field).
4. Leave 1/8" gap at panel edges for expansion.
5. Inspect for "shiners" (missed joists) and correct.
