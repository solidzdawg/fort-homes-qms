# WI-003A: Wall Layout

**Purpose:**
Define the process for accurate wall layout prior to assembly, ensuring all dimensions and openings match plans and code requirements.

## 1. Preparation
- Review approved construction drawings
- Verify floor system is square and level
- Gather required tools: tape, chalk line, square, pencil

## 2. Layout Steps
- Mark all wall locations per plan
- Indicate window and door rough openings
- Mark locations for plumbing/electrical penetrations
- Double-check all dimensions

## 3. Verification
- Supervisor reviews layout before assembly begins
- Corrections made as needed

**Reference:** SOP-003, FORM-I002
