# WI-002A: Joist Layout & Installation

## Step-by-Step Instructions
1. Snap chalk lines for joist layout per engineered plans.
2. Place joists at specified spacing (e.g., 16" o.c.), verify with tape.
3. Install rim boards, blocking, and bridging as required.
4. Secure joists to sill plate with specified fasteners.
5. Install hangers/straps per schedule, verify all nails installed.
