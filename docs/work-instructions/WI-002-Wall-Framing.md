# WI-002: Wall Framing & Sheathing — Visual Work Instructions

---

## Overview

This **Work Instruction** provides a step-by-step visual guide for wall assembly and weatherproofing in **PHASE-2** production. Use this guide alongside **SOP-002** for detailed technical specifications.

**Target Audience:** Framing crew, Bay Supervisors, SIP installation crews  
**Training Required:** Yes (SOP-002 training + hands-on mentoring)  
**Reference:** SOP-002, Engineered Plans, WRB Manufacturer Instructions, Fastener Schedule

## Section 1: Bottom Plate Installation

### Step 1.1: Layout & Fasten Bottom Plates

**Setup Diagram:**

```
        SUBFLOOR (top view)
    
    ┌──────────────────────────────────────┐
    │ Wall Center Line (chalk)             │
    │  ↓                                   │
    │  └─ BOTTOM PLATE (PT 2×4 or 2×6)    │
    │     └─ Adhesive bead (optional)      │
    │     └─ Fasteners: 16d @ 16" o.c.    │
    │        (must hit floor joists)       │
    │                                      │
    │  ← Must hit joist lines below        │
    │                                      │
    │  MARK STUD LOCATIONS: X at 16" o.c. │
    └──────────────────────────────────────┘
```

**What to Do:**
1. Snap chalk line for wall position (perpendicular to floor joists)
2. Lay bottom plate along chalk line; mark stud locations with "X" (16" o.c. typical)
3. Apply ¼" adhesive bead under plate (optional; improves racking resistance)
4. Fasten plate to floor:
   - Drive 16d nails through plate into floor joists below
   - Spacing: 16" o.c. (one fastener per joist)
   - **No nails between joists** (insufficient holding power)
5. Verify plate is level (use 4-foot level):
   - If high/low spots, shim under stud locations
   - Tolerance: ±¼" per 8 feet

**Checklist:**
- [ ] Plate is level (±¼" per 8')
- [ ] All fasteners hit floor joists (verify by inspection below)
- [ ] No gaps between plate and subfloor (shim if needed)
- [ ] Stud marks are visible and spaced per plan

## Section 2: Wall Framing — Stick-Built Method

### Step 2.1: Install Studs

**Stud Installation Detail:**

```
           TOP PLATE
              ↓
    ┌──────────────────┐
    │   ││ ││ ││ ││     │  Studs: 2×4 or 2×6
    │   ││ ││ ││ ││     │  (16" or 24" o.c.)
    │   ││ ││ ││ ││     │
    │   ││ ││ ││ ││     │
    ├────────────────────┤
    │                    │
    │   BOTTOM PLATE     │
    │   (PT 2×4 or 2×6)  │
    └────────────────────┘
    
    FASTENING:
    ┌─────────┐
    │ STUD    │
    │   X     │  ← 2 × 16d nails, toe-nailed
    │   X     │  (one nail per side, angled 10-15°)
    │         │
    └─────────┘
     BOTTOM PLATE
```

**What to Do:**
1. **For each stud location:**
   - Check stud for crown (slight upward curve)
   - Orient stud so crown faces **exterior** (outside)
   - Stand stud upright; align with chalk mark on bottom plate

2. **Fasten stud to bottom plate:**
   - Drive 2 × 16d nails at angle (toe-nail at 10–15°)
   - One nail per side of stud face
   - Alternate sides (prevents stud spinning)
   - Drive nails fully until stud cannot move

3. **Set top plate on studs:**
   - Align top plate with outside edges of studs
   - Studs should be plumb (check with level on stud face):
     - Tolerance: ±¼" per 8 feet height
     - Check at 3 locations per wall (top, middle, bottom of stud)

4. **Fasten studs to top plate:**
   - Same method as bottom: 2 × 16d nails, toe-nailed
   - Ensure stud remains plumb during fastening

**Verification (every 3–5 studs):**
- [ ] Stud is plumb (check with level)
- [ ] Spacing matches mark (±½")
- [ ] Fasteners fully seated (no pops or seating issues)
- [ ] Stud crown faces outward (exterior side)

### Step 2.2: Install Headers & Bracing

**Headers (over windows/doors):**

```
           HEADER (2×6, 2×8, or engineered beam)
              ↓
    ┌──────┬──────────┬──────┐
    │      │          │      │
    │  Stud│ OPENING  │ Stud │  Header beam
    │      │ (window) │      │  fastened to studs
    │      │          │      │
    │      └──────────┘      │
    │         CRIPPLES       │
    │       (2×4 studs)      │
    │         below          │
    └──────────────────────────┘
         BOTTOM PLATE
```

**What to Do:**
1. Size header per engineered plan (depends on span and load)
2. Install header above opening:
   - Support header on full-height studs at each side of opening
   - Header should bear fully on studs (no gaps)
3. Fasten header to studs:
   - Install hardware hangers or plates per plan (Simpson LUS, MUS, or equivalent)
   - Fasten per hardware schedule (typically 2–3 fasteners per hanger)
4. Install cripples (short studs) below opening:
   - Space at 16" o.c. like full-height studs
   - Fasten to sill plate (if present) or header using toe-nails

**Bracing (Lateral Resistance):**

```
    DIAGONAL BRACING (1×4 or 2×4):
    
    ┌─────────────────────┐
    │              ╱╱     │  Bracing runs top-right to bottom-left
    │           ╱╱       │  (or opposite diagonal per plan)
    │        ╱╱          │  
    │     ╱╱             │
    │  ╱╱                │
    │╱╱                  │
    └─────────────────────┘
```

- [ ] Install diagonal bracing if required by plan (prevents racking)
- [ ] Bracing fastened at top and bottom (2 nails per end minimum)
- [ ] OR rely on exterior sheathing OSB (acts as lateral bracing)

## Section 3: Wall Framing — SIP Panel Method

### Step 3.1: Set & Fasten SIP Panels

**SIP Panel Installation:**

```
           TOP SPLINE CONNECTION
                    ↓
    ┌─────────────────────────┐
    │ OSB face (exterior)      │
    │ EPS foam core (insulation)
    │ OSB face (interior)      │
    │                          │
    │  ┌────────────────────┐  │  Typical Panel:
    │  │ Spline groove      │  │  4" or 6" thick
    │  │ on each side       │  │  4' wide, 8'–12' tall
    │  └────────────────────┘  │
    │                          │
    └─────────────────────────┘
         ↓
      BOTTOM PLATE
      (spline grooves routed)
```

**What to Do:**
1. **Prepare base:**
   - Bottom plate must be level and flat
   - Verify spline groove is routed in plate (per SIP manufacturer)

2. **Install spline in bottom plate groove:**
   - Push spline fully into groove
   - Ensure spline is full length of wall section

3. **Set first panel:**
   - Carefully lift panel into position (2–3 people required; panels are heavy)
   - Align panel edge with wall line (use chalk line as guide)
   - Lower panel; spline on panel bottom engages spline in plate

4. **Verify panel is plumb:**
   - Use 6-foot level on panel face
   - Tolerance: ±¼" per 8' height
   - Shim at base if needed

5. **Fasten panel to bottom plate:**
   - Per SIP manufacturer schedule (typically pneumatic nails @ 12" o.c.)
   - Fasteners through panel face into plate

6. **Install spline & set next panel:**
   - Push spline into groove on side of first panel
   - Set second panel; verify plumb
   - Fasten to bottom plate
   - Fasten panels together along spline (per mfg. schedule)

7. **Install top spline & connections:**
   - If installing top plate, push spline into top of panels
   - Install structural hardware per plan (Simpson straps, clips, etc.)

## Section 4: Window & Door Installation

### Step 4.1: Set Window Frame & Unit

**Window Installation Detail:**

```
              SHIMS (cedar)
                 ↓
    ┌──────────────────────────┐
    │ HEADER                   │ ← Pre-assembled header
    │                          │   with trim
    │ WINDOW FRAME ┌────────┐  │
    │              │        │  │
    │ [WINDOW]     │        │  │
    │              │        │  │
    │ WINDOW FRAME └────────┘  │
    │                          │
    │ SILL (sloped out)        │
    └──────────────────────────┘
```

**What to Do:**
1. **Measure rough opening:**
   - Width and height from floor to header
   - Compare against window unit specs

2. **Verify opening is square:**
   - Measure both diagonals
   - Difference must be ≤¼"
   - If out of square, adjust framing

3. **Install window:**
   - Set window frame in opening
   - Use cedar shims on sides and top to level/plumb window
   - Do NOT force window into opening (will cause binding)

4. **Fasten window:**
   - Drive fasteners through window frame nailing flanges into studs
   - Spacing per manufacturer (typically 12" o.c. around frame)
   - Do NOT over-tighten; window frame can warp

5. **Verify operation:**
   - Open and close window smoothly
   - No binding or dragging
   - Latches secure

**Verification:**
- [ ] Window is level (check sill with level)
- [ ] Window is plumb (check sides with level)
- [ ] Diagonals equal (no racking)
- [ ] Window operates smoothly

### Step 4.2: Set Door Frame & Unit

**Door Installation:**

```
              HEADER with structural support
                 ↓
    ┌──────────────────────────┐
    │         [DOOR]           │
    │                          │
    │ Door swings from hinge   │
    │ side (typically left)    │
    │                          │
    │ SHIMS ┌────────────────┐ │ ← 3 per side (top, middle, bottom)
    │       │ DOOR FRAME     │ │
    │       │                │ │
    │       │                │ │
    │       └────────────────┘ │
    │          FLOOR LEVEL     │
    └──────────────────────────┘
    
    CLEARANCE: ≥ ⅛" above flooring (allow for carpet/finish)
```

**What to Do:**
1. **Verify door opening dimensions:**
   - Typical interior: 32"–36" wide, 80" tall
   - Typical exterior: 36" wide, 80" tall
   - Compare against door frame specs

2. **Install door frame:**
   - Set frame in opening; use 3 shims per side (top, middle, bottom)
   - Shims support frame and allow adjustment

3. **Verify frame is plumb & level:**
   - Check both jambs (side pieces) with level
   - Check header (top piece) with level
   - Tolerance: ±⅛"

4. **Verify frame corners are square:**
   - Measure both diagonals across frame opening
   - Diagonals must be equal (within ⅛")

5. **Fasten frame to studs:**
   - Drive 16d nails through jambs into studs
   - Spacing: 12" o.c., minimum 3 per side
   - Nails should pass through shims for best support

6. **Hang door & verify operation:**
   - Install hinges per door manufacturer
   - Hang door in frame
   - Verify door swings freely without binding
   - Check door closes and latches securely

## Section 5: Exterior Sheathing & WRB Installation

### Step 5.1: Install Exterior Sheathing

**Sheathing Application:**

```
           WALL FRAMING (studs @ 16" o.c.)
              ↓
    ┌──────┬──────┬──────┬──────┐
    │      │      │      │      │
    │      │      │      │      │  SHEATHING PANELS
    │      │      │      │      │  (½" or ⅝" OSB/Plywood)
    │      │      │      │      │
    ├──────┼──────┼──────┼──────┤
    │      │      │      │      │  Fastener pattern:
    │      │      │      │      │  12" o.c. edges
    │      │      │      │      │  12" o.c. field (staggered)
    │      │      │      │      │
    └──────┴──────┴──────┴──────┘
    
    STAGGER VERTICAL JOINTS:
    ┌───────┬───────┐
    │ Panel1│ Panel2│  ← Top row
    ├───────┴───────┤
    │ Panel3│ Panel4│  ← Bottom row (offset)
    └───────┴───────┘
```

**What to Do:**
1. **Verify sheathing material:**
   - Grade stamp visible (APA rated)
   - Dry condition (no water staining or swelling)
   - Correct thickness (typically ½" or ⅝")

2. **Layout sheathing:**
   - Arrange panels horizontally (long dimension perpendicular to studs)
   - Stagger vertical joints (offset at least 16" between rows)
   - Leave ⅛" expansion gap at perimeter

3. **Install first panel:**
   - Position at bottom-left corner of wall
   - Align flush with bottom plate edge
   - Leave ⅛" gap at top and sides

4. **Fasten sheathing:**
   - Drive 8d ring-shank nails per fastener schedule:
     - **Edges:** 12" o.c. (perpendicular to studs)
     - **Field:** 12" o.c. in staggered pattern
   - Every fastener **must** hit a stud
   - No shiners (use tap-test to verify)

5. **Continue sheathing installation:**
   - Install remaining panels
   - Ensure vertical joints are staggered
   - Maintain ⅛" gaps at perimeter

**Verification:**
- [ ] Fastener pattern complete (no shiners)
- [ ] Sheathing surface smooth (no soft spots)
- [ ] Joints staggered
- [ ] Gaps at perimeter (⅛")

### Step 5.2: Install Weather-Resistive Barrier (WRB)

**WRB Lap Pattern (Top-Down Installation):**

```
    INSTALLATION DIRECTION (top to bottom)
    ↓
    
    ┌─────────────────────────────┐
    │ Row 1 (top)                 │ ← 6" overlap at horizontal seam
    │                             │
    ├─────────────────────────────┤
    │ Row 2  (overlap ≥ 6")       │
    │                             │
    ├─────────────────────────────┤
    │ Row 3  (overlap ≥ 6")       │
    │                             │
    └─────────────────────────────┘
    
    HORIZONTAL OVERLAP: ≥ 6"
    VERTICAL OVERLAP: ≥ 9"
    (builds lap downward, like roof shingles)
```

**What to Do:**
1. **Prepare substrate:**
   - Sheathing clean and dry
   - Temperature 40°F+ (prevent moisture trapping)

2. **Start at bottom:**
   - Unroll WRB horizontally
   - Align bottom edge at bottom of wall (leave slight sag; do not stretch)
   - Fasten with cap nails:
     - Spacing: 12" o.c. in grid (every 12" horizontally and vertically)
     - Fasteners through caps (not through WRB face alone)

3. **Install second row:**
   - Roll out next section
   - Overlap first row by 6"+ (laps downward)
   - Fasten with cap nails per spacing

4. **Continue to roof line:**
   - Each row overlaps previous by 6"+
   - All horizontal seams lap downward (water sheds)
   - Maintain vertical spacing throughout

5. **Seal all seams:**
   - Use WRB-compatible tape (e.g., Tyvek Seam Tape)
   - Tape all horizontal and vertical seams
   - Press firmly; ensure full contact

**Checklist:**
- [ ] WRB installed bottom-up (laps downward)
- [ ] Horizontal overlap ≥ 6"
- [ ] Vertical overlap ≥ 9"
- [ ] All seams taped (no exposed laps)
- [ ] No tears or gaps
- [ ] Fastener spacing 12" o.c. in grid

### Step 5.3: Install Window/Door Flashing

**Flashing Sequence (Window Example):**

```
                WRB (weather-resistive barrier)
                    ↓
    ┌───────────────────────────────────────┐
    │           HEAD FLASHING               │ ← Installed ABOVE window
    │     (slopes outward & downward)       │
    │                                       │
    │  ┌─────────────────────────────────┐  │
    │  │                                 │  │
    │  │      WINDOW OPENING             │  │
    │  │                                 │  │
    │  │  JAMB FLASHING (both sides)     │  │
    │  │         ↓                       │  │
    │  │ ┌──────────────────────────┐   │  │
    │  │ │  WINDOW UNIT             │   │  │
    │  │ │  (installed in opening)  │   │  │
    │  │ └──────────────────────────┘   │  │
    │  │                                 │  │
    │  │         SILL FLASHING            │  │
    │  │    (slopes outward & down)      │  │
    │  └─────────────────────────────────┘  │
    │                                       │
    └───────────────────────────────────────┘
```

**What to Do:**

1. **Install head (top) flashing:**
   - Apply metal flashing or flashing tape above window frame
   - Flashing slopes downward and outward (prevent water pooling)
   - Extend 2–3" beyond window width on each side
   - Tape side edges to WRB with sealing tape

2. **Install jamb (side) flashing:**
   - Apply flashing tape down both sides of opening
   - Extend from header to sill
   - Overlap WRB behind flashing
   - Tape seal at edges

3. **Install sill (bottom) flashing:**
   - Install sill pan or flashing tape under window sill
   - Slope downward and outward
   - Extend beyond opening width on sides
   - Ensure weeps (drainage holes) are clear

4. **Seal all flashing:**
   - Use WRB-compatible tape at all seams
   - Seal where flashing meets WRB
   - Seal where flashing meets window frame
   - **Test:** Spray water over flashing; water should run down and out

**Verification:**
- [ ] Head flashing slopes downward
- [ ] All flashing extended 2–3" beyond opening
- [ ] All seams taped and sealed
- [ ] Weeps clear (if applicable)
- [ ] Water test passes (spray water; no penetration)

## Section 7: Troubleshooting

### Problem: Stud Not Plumb
**Solution:** Check for twisted lumber; if severe, replace. If minor, shim at top or bottom plate using cedar shims.

### Problem: Opening Out of Square
**Solution:** Measure diagonals; if difference >¼", adjust frame corners with shims or re-frame if structural.

### Problem: WRB Torn or Gap
**Solution:** Patch immediately with WRB-compatible tape. If tear >2 sq. inches, consider panel replacement.

### Problem: Water Observed Behind Flashing
**Solution:** Identify leak source. Re-tape flashing seams. Verify flashing slopes downward. Consider caulk sealant if gaps exist.
