# WI-003B: Sheathing Installation

**Purpose:**
Standardize the installation of wall sheathing for structural integrity and code compliance.

## 1. Preparation
- Confirm wall framing is complete and square
- Select correct sheathing material per plans
- Gather fasteners, adhesives, and tools

## 2. Installation Steps
- Apply adhesive to studs (if required)
- Place sheathing panels with staggered joints
- Fasten per schedule (nailing pattern, edge distance)
- Leave required expansion gaps
- Mark all hold points for inspection

## 3. Inspection
- Supervisor or QA inspects nailing, panel fit, and fastener type
- Corrections made before proceeding

**Reference:** SOP-003, FORM-I002
