## WORK INSTRUCTIONS — PHASE 1: FLOOR SYSTEM

### Step 1: Floor Truss Layout & Installation

**Purpose:** Establish the structural floor deck per CC21TE structural drawings.

**Parts/Materials:**
- 9-1/4" pre-engineered floor trusses (qty per layout)
- 1-3/4" × 9-1/4" LVL outer rim board
- 2×10 dimensional lumber inner rim board
- (3) 1-3/4" × 9-1/4" LVL floor beam (at grid line B)
- BC-T1 beam connections (4 total)

**Tools Required:**
- Framing nailer (0.131" capacity)
- Tape measure, chalk line, laser level
- Impact driver with SDWS bit

**Setup:**
1. Position chassis in production bay
2. Verify chassis level within ±1/4" across diagonal
3. Verify squareness: diagonals within ±1/4"
4. Mark truss layout at 16" O.C. on rim boards

**Detailed Actions:**

| Sub-Step | Action | Fastener/Hardware |
|----------|--------|-------------------|
| 1.1 | Install LVL outer rim boards along long walls | (3) 0.131" × 3" nails @ 6" O.C. |
| 1.2 | Install LVL outer rim boards along short walls (no splices) | (3) 0.131" × 3" nails @ 6" O.C. |
| 1.3 | Install 2×10 inner rim behind LVL | (7) 0.131" × 3" nails per member, staggered |
| 1.4 | At rim corners, install (3) SDWS22800DB screws | Screws at each corner |
| 1.5 | Install (3)-ply LVL floor beam at Grid B | SDWS22500DB @ 12" O.C. ea. side, 6" stagger |
| 1.6 | At LVL splice locations, add 2×10 block + (4) SDWS22400 each side | (8) screws per splice |
| 1.7 | Install floor trusses at 16" O.C. layout marks | Per truss manufacturer |
| 1.8 | Install BC-T1 beam connections at floor beam bearing | (6) Simpson A34 with 0.131" × 2-1/2" nails |

**In-Process Checks:**
- [ ] Truss spacing verified at 16" O.C. ±1/2"
- [ ] Rim board level within ±1/8" per 8'
- [ ] All LVL splices have (8) SDWS screws installed
- [ ] Beam connections BC-T1 complete with all hardware
- [ ] No truss damage (cracks, splits, connector plate lift)

**Safety Notes:**
- Do not stand on trusses until blocking is installed
- Verify all nail gun connections before operation
- Wear hearing protection during nailing

**Common Mistakes to Avoid:**
- ❌ Splicing LVL where not permitted (between Grids 1-2 or 3-4)
- ❌ Missing SDWS screws at rim corners
- ❌ Trusses installed with crown down
- ❌ Insufficient nail penetration into LVL

### Step 2: Subfloor Sheathing Installation

**Purpose:** Install structural floor deck per decking schedule D-1.

**Parts/Materials:**
- 23/32" (3/4" nominal) T&G APA-rated plywood or OSB
- Subfloor construction adhesive
- 0.131" × 2-1/2" ring-shank nails

**Tools Required:**
- Framing nailer, adhesive applicator
- Tape measure, chalk line
- Circular saw for field cuts

**Setup:**
1. Verify all trusses are secured
2. Clean top of trusses (remove debris, sawdust)
3. Snap reference lines for panel layout
4. Plan staggered joint pattern (min 4' offset)

**Detailed Actions:**

| Sub-Step | Action | Specification |
|----------|--------|---------------|
| 2.1 | Apply continuous bead of adhesive to truss tops | 1/4" bead, no gaps |
| 2.2 | Position first panel at corner, tongue side out | Align with rim |
| 2.3 | Press panel firmly until adhesive squeeze-out visible | Full contact |
| 2.4 | Fasten panel edges | 0.131" × 2-1/2" nails @ 6" O.C. |
| 2.5 | Fasten panel field | 0.131" × 2-1/2" nails @ 12" O.C. |
| 2.6 | Continue with staggered pattern, min 4' joint offset | — |
| 2.7 | Leave 1/8" expansion gap at panel edges | Per manufacturer |
| 2.8 | Cut and fit panels around penetrations | Mark per floor plan |

**In-Process Checks:**
- [ ] Adhesive squeeze-out visible at all joints
- [ ] Edge nailing at 6" O.C. verified with gauge
- [ ] Field nailing at 12" O.C. verified
- [ ] No voids under panels (tap test)
- [ ] Panel joints staggered minimum 4'
- [ ] Expansion gaps present at perimeter

**Safety Notes:**
- Keep walking surface free of debris
- Mark all penetration locations clearly before cutting
- Use proper blade guard when cutting panels

**Common Mistakes to Avoid:**
- ❌ Insufficient adhesive coverage
- ❌ Nails driven through panel face (nail set too deep)
- ❌ Panels not fully seated before nailing
- ❌ Missing expansion gaps causing buckle risk

- SIP panels are heavy; use proper lifting techniques
- Brace panels immediately after setting
- Keep foam sealant away from ignition sources

**Common Mistakes to Avoid:**
- ❌ Using OSB splines within 2' of openings (lumber required)
- ❌ Missing foam sealant at joints (air barrier breach)
- ❌ LVL cap splices aligned with 2×6 plate splices
- ❌ SIP screws not achieving 1-1/2" min penetration
