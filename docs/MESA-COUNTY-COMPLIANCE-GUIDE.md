### 7.2 Required Documents for Permit Application

| Document | Purpose | Required By |
|:---|:---|:---|
| Building Permit Application | Official request for permit | All jurisdictions |
| Site Plan | Show home location, setbacks, utilities | All jurisdictions |
| Foundation Plan | Engineering drawings for foundation | All jurisdictions |
| Manufacturer's Data Sheet | HUD label info, specs | All jurisdictions |
| Installer Certification | Proof of licensed installer | State requirement |
| Anchoring Plan | Wind Zone II tie-down plan | All jurisdictions |
| Septic Permit | If not on sewer system | Mesa County Health |

### 7.3 Typical Permit Fees

| Fee Type | Approximate Cost | Notes |
|:---|:---:|:---|
| Building Permit | $500-$1,500 | Based on home value |
| Plan Review | $100-$300 | May be included in building |
| Electrical Permit | $100-$200 | For utility connection |
| Plumbing Permit | $100-$200 | For water/sewer connection |
| Mechanical Permit | $75-$150 | For HVAC if applicable |
| Septic Permit | $200-$400 | If not on sewer |
| Impact Fees | Varies | Some jurisdictions only |

*Note: Fees subject to change. Verify with jurisdiction.*
