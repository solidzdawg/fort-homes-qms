## Key Schedules Inventory

### Architectural Schedules (AM10)
1. **Door Schedule** - Marks (01-05), width, height, count, remarks
2. **Window Schedule** - Marks (A-E), width, height, count, remarks  
3. **Light and Vent Required/Provided** - Bedroom, bathroom, kitchen/living requirements

### Structural Schedules (SM10, SM11)
1. **Header Schedule** - Marks (H1, H2, H3), size, detail reference
2. **Wall Schedule** - Wall type, properties, code references
3. **Wood I-Joist Strapping Schedule** - Marks (14C1, 14C2), stud types, quantities, strap types, capacities, details

### Mechanical Schedules (M02)
1. **Heat Pump Indoor Unit Schedule** - Manufacturer, model, air flow rate, nominal cooling capacity, nominal heating capacity, power
2. **Heat Pump Outdoor Unit Schedule** - Manufacturer, model, air flow rate, net weight (lbs), capacity, nominal heating capacity, power
3. **Exhaust Fan/Heater Schedule** - Manufacturer, model
4. **Air Terminal Schedule** - Mark, manufacturer, location, model, qty
5. **Air Exchanger Ventilator Schedule** - Mark, manufacturer, model, sensible airflow (cfm), net weight (lbs), primary supply opening, primary supply opening

### Plumbing Schedules (P03)
1. **Water Heater Schedule** - Mark, manufacturer, voltage
2. **Plumbing Fixture Schedule** - Mark, description, manufacturer/model, model, trim, fixture connections, notes
3. **Water Supply Fixture Unit Calculation** - Location, fixture or group, hot, cold, combined

### Electrical Schedules (E02)
1. **Panel Schedule** - Circuit description, trip, poles, wire size (main panel and subpanel)
2. **Lighting Fixtures Schedule** - Mark, description
3. **Dwelling Unit Service Calculation** - Detailed load calculations
